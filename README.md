# Chatapplikasjon

## Datamodell

Chat-appen brukar ein enkel og strukturert datamodell i **MySQL**.  
Ho inneheld tabellar for brukarar, vener, meldingar og grupper, der alt heng saman gjennom framandnøklar.  
Modellen er bygd for å vere lett å utvide, slik at nye funksjonar som bilete, reaksjonar eller varslingar kan leggjast til utan store endringar.
